# Stack 
- Database : MongoDB
- Frontend : React Redux Typescript
- Backend : ExpressJS

# Features 
- Authentication with JWT/RefreshJWT
- CRUD API to manage basket and products

# Starting the project
- Start database : 
  - cd docker
  - docker compose up -d
    
- Start Frontend :
  - cd Frontend
  - npm install && npm run dev

- Start Backend
- cd Backend
- nodemon
